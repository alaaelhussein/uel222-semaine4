Purpose
This repository contains a Symfony 6.4 mini-project for UE L222 week 4. It includes CRUD for articles and categories, a Twig-based dynamic menu, and Bootstrap styling.

----------------------------------------
Stack
PHP 8.1, Symfony 6.4, Twig, Doctrine ORM, SQLite (default dev database), and Bootstrap 5.3.8 (local assets).

----------------------------------------
Run
Install PHP dependencies with Composer and run the application with a local PHP server or Symfony CLI. Apply database migrations and load fixtures before first use.
